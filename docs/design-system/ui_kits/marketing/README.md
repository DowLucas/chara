# Chara — Marketing site UI kit

A single-page marketing site at 1280px design width. Static HTML — the layout vocabulary is fully encoded in CSS, no React needed.

## Sections
1. **Nav** — wordmark + 3 links + CTA
2. **Hero** — split heading with vermillion accent, paired with a live-looking app fragment
3. **Features** — three-column numbered ledger
4. **How it works** — three steps with rule-line headers
5. **Quote** — large pull quote, sentence case
6. **CTA band** — dark coffee on sand dune with the CHARA stamp
7. **Footer** — single line, low-contrast

## Notes
- The dark CTA band is the only inversion in the system; it's where the brand permits the dark palette outside dark mode.
- Hero numbers and feature numerals (`01 / 02 / 03`) use mono — they read as ledger row numbers.
- No images, no illustrations, no icons in this surface. The brand carries the page.
